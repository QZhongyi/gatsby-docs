## Netlify
Netlify 是一个提供**静态资源托管**的综合平台

- 托管静态资源，并自动启用 CDN 加速
- 支持 CI
- 能绑定自定义域名
- 提供免费 TLS 证书，启用 HTTPS


将 Netlify 项目和 Github 仓库关联后，只要 push 了代码，Netlify 就会自动根据配置进行构建并部署。相比 Github Pages，启用了 CDN 的 Netlify 托管站点在国内请求速度快会很多
